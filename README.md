# 🛰️ NEOwatch Bot

Telegram бот для відстеження космосу: астероїди, МКС, запуски ракет, полярне сяйво та екіпаж.

## 🚀 Функціонал

- 🌑 **Астероїди** — дані поблизу Землі з NASA API
- 🛰️ **МКС** — положення та проходження над твоєю локацією
- 📡 **Персональна локація** — кожен вказує своє місто
- 🌌 **APOD** — Фото дня від NASA
- 🚀 **Запуски ракет** — найближчі запуски SpaceX/NASA
- 👨‍🚀 **Екіпаж МКС** — хто зараз на станції
- 🛰️ **Starlink** — ланцюжок супутників над тобою
- 🌌 **Космопогода** — прогноз полярного сяйва
- 🔔 **Автоматичні сповіщення** — МКС та фото дня
- 💾 **SQLite база** — збереження даних користувачів

## 🆕 Нові функції

### 🚀 Запуски ракет
Дата, час, місце, статус запуску від SpaceX, NASA та інших

### 👨‍🚀 Екіпаж МКС
Список астронавтів на станції прямо зараз + загальна кількість людей в космосі

### 🛰️ Starlink
- Ланцюжок супутників SpaceX над твоєю локацією
- Видно як рухомі «зірки» в ряд
- Ідеально для спостережень в темну пору

### 🌌 Космопогода (полярне сяйво)
- Індекс геомагнітної активності Kp
- Прогноз можливості бачити полярне сяйво
- Україна бачить при Kp 7+

## 📍 Як вказати локацію

1. Натисни кнопку <b>"📍 Вказати місто"</b>
2. Напиши назву (Kyiv, Lviv, Odesa...)
3. Бот автоматично знайде координати

## 🔔 Автоматичні сповіщення

| Тип | Коли | Що робити |
|-----|------|-----------|
| 🛰️ МКС | 10 хв до проходження | Натисни ⚙️ і включи "✅ Сповіщення МКС" |
| 🌌 APOD | Щодня о 9:00 Kyiv | Натисни ⚙️ і включи "✅ Фото дня NASA" |

## 🔌 API

- **NASA API** — астероїди та фото дня
- **N2YO API** — трекінг МКС та супутників
- **Launch Library 2** — запуски ракет
- **NOAA Space Weather** — космопогода
- **Open Notify** — екіпаж МКС
- **OpenStreetMap Nominatim** — геокодування міст
