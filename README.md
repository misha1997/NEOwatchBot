# 🛰️ NEOwatch Bot

Telegram бот для відстеження астероїдів, космічних супутників і щоденного фото дня від NASA.

## Функціонал

- 🌑 **Астероїди** — дані про близькі до Землі об'єкти з NASA API
- 🛰️ **МКС** — поточне положення та проходження над твоєю локацією
- 📡 **Глобальний доступ** — кожен користувач вказує своє місто
- 🌌 **APOD** — фото дня від NASA
- 🔔 **Автоматичні сповіщення** — МКС та фото дня
- 💾 **SQLite база** — збереження даних користувачів

## 📍 Персональна локація

**Як вказати своє місто:**
1. Натисни кнопку з локацією (наприклад 📍 Вказати місто)
2. Напиши назву міста (англійською або українською)
3. Бот автоматично знайде координати

**Що це дає:**
- Точні прогнози проходження МКС твоїм небом
- Швидше отримання сповіщень
- Персоналізовані дані

## Автоматичні сповіщення

**🛰️ МКС:**
- Бот перевіряє кожні 5 хвилин
- Сповіщення за 10 хв до проходження
- Індивідуально для кожної локації

**🌌 APOD:**
- Щодня о 9:00 Kyiv time
- Для всіх підписаних

## Структура бази даних

```
SQLite база: neowatch.db

Таблиці:
├── users (user_id, chat_id, city, lat, lon, 
│          subscribed_iss, subscribed_apod,
│          last_iss_pass, last_apod_date)
└── iss_passes (user_id, pass_time, duration, max_elevation)
```

## Встановлення

```bash

# Встановити залежності
```

## API

- NASA NEO API — астероїди
- NASA APOD API — фото дня
- N2YO API — трекінг МКС
- OpenStreetMap Nominatim — геокодування міст
